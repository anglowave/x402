x402 Gateway

Setup Instructions

Python Setup Required: Your Python installation needs pip to run the gateway server. Follow the steps below to fix this.

1

Install/Repair Python with pip

Download Python from python.org and make sure to check "Add Python to PATH" and "Install pip" during installation.

2

Verify pip is installed

Open PowerShell and run:

python -m ensurepip --default-pip

Then verify:

python -m pip --version
3

Install gateway dependencies

Navigate to the gateway folder and install:

cd C:\Users\7haid\OneDrive\Desktop\X402\gateway
python -m pip install flask flask-cors requests
4

Start the gateway server

python server.py

Then open http://localhost:3000

Alternative: Use the Python from x402 project
If you already have Python with pip somewhere else, you can use it:

cd C:\Users\7haid\OneDrive\Desktop\X402\gateway
C:\path\to\your\python.exe -m pip install flask flask-cors requests
C:\path\to\your\python.exe server.py
